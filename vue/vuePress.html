# VuePress

# Installation

  1. Create project folder & initialize
# ./projectName
npm init
  1. Install VuePress locally
npm install -D vuepress
  1. Add run scripts to package.json
"scripts": {
  "docs:dev": "vuepress dev docs",
  "docs:build": "vuepress build docs"
}

/* npm run docs:dev */

# Directory Structure

  • A folder in docs for each topic.
    • index.md = topic landing page
    • additional pages as needed
  • .vuepress folder holds config files (opens new window) and assets
    • config.js defines general settings and routes
.
├── docs/
│   ├── index.md
│   │ 
│   ├── .vuepress/
│   │   ├── config.js
│   │   │
│   │   ├── components/
